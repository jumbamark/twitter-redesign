# Twitter Redesign

A project that tests Django and React knowledge through a twitter-like web application that allows users to sign in, create and like tweets and view the profiles of other users

## Build With

- Python v3.8.10
- Django v4.0.2
- DjangoRestFramework
- React
- Bootstrap v5.1
- PostgresQL
